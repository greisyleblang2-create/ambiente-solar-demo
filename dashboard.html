Einrichtungsanleitung – Google Sheets + n8n

1

Google Spreadsheet erstellen

Öffne sheets.google.com → Neue Tabelle → Name: "Ambiente Solar Leads"
Erste Zeile genau so ausfüllen (jede Spalte einzeln):

A: zeitstempel B: typ C: vorname D: nachname E: email F: telefon G: anlagentyp H: nachricht I: antworten J: status

Dann: Datei → Im Web veröffentlichen → Gesamtes Dokument → CSV → Veröffentlichen
CSV-Link kopieren → unten in diesem Dashboard bei SHEET_CSV eintragen.

2

n8n Workflow A – Neue Anfrage empfangen

In n8n Cloud: Neues Workflow erstellen → "Ambiente Solar – Neue Anfrage"

Node 1:Webhook · Methode: POST · Path: ambiente-solar-leads · Auth: None · Aktiv schalten → URL kopieren
Node 2:Google Sheets → Append Row · Spreadsheet: Ambiente Solar Leads · Sheet: Sheet1

Spalten-Mapping im Google Sheets Node:

zeitstempel ← {{ new Date().toLocaleString('de-DE') }} typ ← {{ $json.body.typ }} vorname ← {{ $json.body.vorname }} nachname ← {{ $json.body.nachname || '' }} email ← {{ $json.body.email }} telefon ← {{ $json.body.telefon || '' }} anlagentyp ← {{ $json.body.anlagentyp || '' }} nachricht ← {{ $json.body.nachricht || '' }} antworten ← {{ JSON.stringify($json.body.antworten || []) }} status ← Neu

→ Webhook-URL nach Aktivierung in index.html bei WEBHOOK_URL eintragen.

3

n8n Workflow B – Status aktualisieren

Damit die Aktionsbuttons im Dashboard den Status in Google Sheets ändern:

Node 1:Webhook · POST · Path: ambiente-solar-update
Node 2:Google Sheets → Get Rows · alle Zeilen lesen
Node 3:Code Node · Zeile mit passender E-Mail finden, Row-Index zurückgeben
Node 4:Google Sheets → Update Row · Status-Spalte J auf neuen Wert setzen

→ Webhook-URL unten in diesem Dashboard bei WEBHOOK_UPDATE eintragen.

4

URLs in den Dateien eintragen

📄 dashboard.html · Zeile 1 in <script>: SHEET_CSV = Google Sheets CSV-Link

📄 dashboard.html · Zeile 2 in <script>: WEBHOOK_UPDATE = Workflow-B-Webhook-URL

📄 index.html · WEBHOOK_URL = Workflow-A-Webhook-URL (z.B. https://gleblang.app.n8n.cloud/webhook/ambiente-solar-leads)
